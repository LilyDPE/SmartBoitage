# Components

This directory contains reusable React components for the SmartBoitage application.

## Components List

- `Map.tsx` - Leaflet map component with geolocation support
- `Navbar.tsx` - Navigation bar component
- `NavigationGuide.tsx` - Turn-by-turn navigation with voice guidance
- `ProtectedRoute.tsx` - Route protection wrapper
- `SegmentList.tsx` - Segment list display component
- `ServiceWorkerRegistration.tsx` - PWA service worker registration
- `SessionProvider.tsx` - NextAuth session provider wrapper
- `TourPlayer.tsx` - Tour playback component
- `ZoneDrawer.tsx` - Zone drawing component
